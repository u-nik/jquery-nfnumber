jQuery NFNumber Plugin

Dieses Plugin ermöglicht die einfache Formatierung und Validierung von Werten in Formularfeldern, die zur Eingabe von Numerischen Werten verwendet werden.

Raw

1) Standard. Keine nennenswerten Beschränkungen. Deutsches Zahlenformat.

2) Schrittgröße von 0,005, Genauigkeit von 3 und dem € Suffix.

3) Schrittgröße automatisch anhand der Genauigkeit bestimmt, Multiplikator ist 10000, ¥ Prefix, Englisches Format

4) Bereich eingeschränkt auf -150 bis -100.

Formatted

5) Standard. Keine nennenswerten Beschränkungen. Deutsches Zahlenformat.

Shortcuts bei fokusiertem Eingabefeld

Pfeil hoch oder "+" (Shift)
Schritt hochzählen (Großer Schritt)
Pfeil runter oder "-" (Shift)
Schritt runterzählen (Großer Schritt)
Leertaste (Shift)
Auf MIN setzen (Auf MAX setzen)
